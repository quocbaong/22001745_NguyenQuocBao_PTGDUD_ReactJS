# Repo: Bài tập thực hành theo tuần — PTGDUD ReactJS

Mô tả

Repo này chứa các bài tập thực hành theo từng tuần của môn PTGDUD (ReactJS). Mỗi tuần được tổ chức thành thư mục hoặc các file HTML/JS tương ứng, dùng để minh họa các bài tập, ví dụ và demo đơn giản.

Cấu trúc thư mục chính

- `WEEK1/` — Bài tập tuần 1 (ví dụ: `Cau1.html`, `Cau2.html`, `Cau3.html`, `Cau4.html`, ...)
- `WEEK1/MinhChung/` — Hình ảnh minh chứng cho các bài tập

Cách chạy

- Mở trực tiếp các file `.html` bằng trình duyệt (kéo thả hoặc chuột phải -> Open with...)
- (Tùy chọn) Dùng extension Live Server trong VS Code để chạy và tự động reload

Quy ước

- Tên file theo dạng `CauX.html` cho bài tập X trong tuần
- Các biến, hàm trong file viết rõ ràng để dễ theo dõi kết quả trên console hoặc giao diện

Ghi chú

- Đây là repo học tập; các bài tập tập trung vào JavaScript/HTML cơ bản và sẽ được phát triển dần theo các tuần học.
